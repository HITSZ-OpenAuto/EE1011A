## 课程评价

课程主要内容涵盖《电路理论基础（第四版）》前 9 章：
1. **电路元件和电路基本定律**：电路模型、基尔霍夫定律、元件时域模型等。
2. **线性直流电路**：电阻网络等效、支路电流法、回路电流法、节点电压法、含运放电路分析。
3. **电路定理**：叠加定理、戴维南/诺顿定理、特勒根定理、互易定理、对偶原理。
4. **正弦稳态电路的相量分析**：相量法、阻抗导纳、功率、耦合电感、理想变压器。
5. **三相电路**：对称/不对称三相电路计算、功率。
6. **非正弦周期电流电路**：谐波分析、有效值与平均功率。
7. **频率特性和谐振现象**：网络函数、串联/并联谐振。
8. **线性动态电路暂态过程的时域分析**：一阶/二阶电路响应、三要素法、卷积积分、状态变量法。
9. **线性动态电路暂态过程的复频域分析**：拉普拉斯变换应用、复频域网络函数。

> 文 / [Oliver Wu]([https://github.com/OliverWu515](https://github.com/OliverWu515)), 2024.2

## 其他信息

### 学习顺序建议

我个人会将第二、三章的学习顺序调整如下：
2.3 支路电流法；2.4 回路电流法；2.5 节点电压法；2.6-2.7 含运放电路的分析；2.1 电阻网络等效；3.1/2.2/3.3 含源支路和含源一端口等效；3.2 齐性定理、叠加定理；3.4-3.6 特勒根定理、互易定理、对偶原理。

此外，预习时可将第六章（非正弦周期电流电路）提到第五章（三相电路）之前。
